# MyBestBroker

MyBestBroker è una web app interattiva sviluppata in **Streamlit** per confrontare
le **commissioni di trading** applicate da diversi broker o banche.

L’obiettivo non è suggerire un broker “migliore in assoluto”, ma aiutare l’utente
a valutare quale sia **più conveniente in base al proprio stile operativo**.

---

## Funzionalità principali

- Confronto commissioni per:
  - importo operazione
  - asset class
- Supporto a più broker configurabili
- Commissioni:
  - minimo
  - percentuale
  - massimo (opzionale)
- Classifica sul singolo trade
- Fasce di convenienza aggregate
- Grafico comparativo
- Consigliatore annuale (stima costo annuo)

---

## Come funziona

1. Inserisci l’importo medio delle operazioni
2. Seleziona un’asset class
3. Configura i broker e le commissioni
4. Analizza i risultati e confronta i costi

---
